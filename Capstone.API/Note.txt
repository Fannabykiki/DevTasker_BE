To run migration to your database:

1. Change connection string in appsettings.json 

2. Open Package Manager Console

3. Run migration:
add-migration CreateDatabase -context CapstoneContext -o Migrations
dotnet ef migrations add AddFKStatus

4. Update migration:
update-database
